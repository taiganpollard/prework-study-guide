HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements work togerther to build the structure of a wbepage.
  • The div element creates a division in a page.
  • The p element creates a paragraph.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Text-align allows you to center a title, header or text on a page.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git push origin feature/branch-name: pushes up all changes
  • git pull origin main: makes sure that everything is up to date before doing a pull request
  • git commit -m "description": commit changes to repository
  • git add -A: stage changes

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The const keyword allows variable to be created where values can't be reassigned
  • If statements allow you to introduce decision-making into codes.
